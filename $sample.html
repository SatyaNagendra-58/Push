Glossary
Avgerage Response Time

Response time testing is a measurement of the amount of time that passes between a user's request and a response from the server, application, website or device.

Hits/Second

Hits per second defines the total load set by the virtual users on the server.

Error

Error means how many requests failed or resulted in error throughout of the test duration.

95% Response Time

It represents 10% of the users have exceeded time to reach the server.

90% Response Time

For example, the 90th percentile (abbreviated as p90) indicates that 90% of the sample is below that value and the rest of the values (that is, the other 10%) are above it.
